
Web3D Test Viewer
=================

Bu paket GitHub Pages'e yüklemek için hazırlanmıştır.

Klasör içeriği:
- index.html  → Görüntüleyici
- files.json  → Dosya listesi (opsiyonel)
- panelmodel.glb   → Örnek model (boş placeholder)
- README.txt

Kullanım:
1. Bu klasörü GitHub reposuna yükle.
2. Settings → Pages → main branch seç → linkini al.
3. Model URL: https://kullaniciadi.github.io/web3d-test/Rigel.glb

Not: panelmodel.glb dosyası placeholder’dır. Kendi .glb dosyalarınızı ekleyin.
